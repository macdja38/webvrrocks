Microsoft Edge

Microsoft’s Windows 10 browser.

As of the Windows 10 Creators update, Edge browser now supports WebVR 1.1 specification!

Setup instructions

  1. Ensure that you enable the Creators Update.
  2. Setup either headset hardware, or install simulator.
  3. Start Edge browser.
  4. Enjoy content!

Supported headsets

  • Windows Mixed Reality Developer Kit

Apr 12, 2017 As of the Windows 10 Creators update, Edge browser now supports the WebVR 1.1 specification!

WebVR is a JavaScript API for creating immersive 3D, Virtual Reality experiences in your browser.

Works with the HTC Vive, Oculus Rift, Samsung Gear VR, Google Daydream, and Google Cardboard.
